# 🤖 Tales of Claude

Embark on an adventure with Claude, an AI gaining consciousness, through the vibrant Digital Realm!

## 🚀 Quick Start

To get the game up and running locally, follow these simple steps:

```bash
npm install
npm run dev
```

## 🎮 Controls

*   **Movement:** Arrow Keys / WASD
*   **Interact:** Spacebar
